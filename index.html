All the steps required to change careers

I am currently starting my journey to change careers. I have always loved
computers but never focused on them as a career. After 30 years I realized
how important it is to be happy in what you do and decided I needed a change.
It is now 09/01/21 and I am relearning from scratch at HTML. I will update
as I go with struggles and successes to hopefully help someone else along the way!

Goals

  • Learn HTML
  • Learn CSS
  • Learn Javascript
  • Learn Backend properties
  • Learn Python
  • Learn Ios Development
  • Learn Unity/Blender to create mobile games for passive income

Estimated Timeline

  • Take 2 months to learn HTML

  • Take 3 months to learn Javascript
